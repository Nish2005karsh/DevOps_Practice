What s  Jenkins?
CI/CD stands for continuous integration/continuous deployment(delivery)
Pipeline is group of instructions
CI (Continuous Integration): Automatically building and testing your code every time developers push changes to a shared repository.
CD (Continuous Delivery/Deployment): Automatically deploying the code to staging or production environments after successful tests.
