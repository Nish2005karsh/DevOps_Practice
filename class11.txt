Kubernetes
Commands

1.Cluster Information

kubectl version //Show client and server version
kubectl cluster-info //Display cluster info
kubectl get nodes //List of nodes in a cluster
kubectl describe node <node>  //Show detailed info about a node
kubectl get namespace //List all namespaces

2.Pod Management

kubectl get pods  //List all pods
kubectl get pods -A //List pods in all namespaces
kubectl describe pod <pod-name> //Detailed info about a pod
kubectl logs<pod-name> //Show logs from a pod
kubectl exec -it <pod-name> --bash //Access pod shell
kubectl delete pod <pod-name>  //Delete a specific pod

3.Deployment Management

kubectl get deployments //List all deployments
kubectl create deployment nginx --image=nginx //Create a simple deployment
kubectl describe deployment nginx //Show detailed deployment
kubectl scale deployment nginx --replicas=3  # Scale up/down pods
kubectl rollout status deployment nginx      # Check deployment rollout status
kubectl delete deployment nginx              # Delete a deployment

4. Service Management

kubectl get services                    # List all services
kubectl expose deployment nginx --type=NodePort --port=80  # Expose deployment
kubectl describe service nginx           # Show service details
kubectl delete service nginx             # Delete a service

5. Configurations & Files
kubectl apply -f deployment.yaml       # Apply a YAML config
kubectl delete -f deployment.yaml      # Delete resources defined in YAML
kubectl get all                        # List all resources in current namespace

6.Namespace Operations


kubectl create namespace dev
kubectl get namespaces
kubectl config set-context --current --namespace=dev  # Switch namespace
kubectl delete namespace dev


7. Debugging & Troubleshooting

kubectl describe pod <pod-name>         # Details & events of a pod
kubectl logs <pod-name> --previous      # Logs from crashed pod
kubectl top pod                         # Show resource usage (needs metrics server)
kubectl get events --sort-by=.metadata.creationTimestamp  # View events

8. Node & Cluster Maintenance

kubectl drain <node-name>               # Safely remove node from cluster
kubectl cordon <node-name>              # Mark node unschedulable
kubectl uncordon <node-name>            # Mark node schedulable again

9. Contexts & Config


kubectl config get-contexts
kubectl config current-context
kubectl config use-context <context-name>


10. Useful Shortcuts
kubectl get po,svc,deploy   # List pods, services, and deployments together
kubectl get all -n kube-system  # Get all resources in system namespace
kubectl explain pod         # Explain what a Pod is
